# Windrose
Embark on a PvE survival adventure in the Age of Piracy. Fight on land and sea, solo or with friends. Build, craft and explore vast open world filled with dark secrets. Master soulslite combat and take on challenging bosses, command your ship and plunder unspoken treasures!
## Server Requirements

- Minimum RAM: 8GiB
- Minimum Disk: 5GiB

# Connecting to the server
Players can connect by
- Using the InviteCode provided by the server (and password if set)
- Using Direct Connect and entering the IP:Port and password if set
- The InviteCode can be found and changed in /home/container/R5/ServerDescription.json

## Server Ports
| Port            | Default |
| --------------- | ------- |
| Game            | 7777    |
| Query           | 7778    |
